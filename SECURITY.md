# Security Policy

## Supported versions

Only the latest minor version of QuantedAgents receives security updates.

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **d.forino@quanted.com** with:

- A description of the vulnerability.
- Steps to reproduce (a minimal repro case is ideal).
- The affected version(s) of `quanted-agents`.
- The potential impact, as you see it.

We will acknowledge receipt within 3 business days and aim to release a patch within 14 days for confirmed vulnerabilities. We will coordinate disclosure with you and credit the reporter unless requested otherwise.
